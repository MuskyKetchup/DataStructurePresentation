Стек

Стек — абстрактный тип данных, представляющий собой список элементов, организованных по принципу LIFO

Очередь

Очередь - абстрактный тип данных с дисциплиной доступа к элементам FIFO.

Дек

Дек — структура данных, представляющая из себя список элементов, в которой добавление новых элементов и удаление существующих производится с обоих концов.

Стек и куча

При выделении памяти в контексте операционной системы обычно разделяют "виды" памяти на два:
  • Стек(stack)
  • Куча(heap)

Разница между в том, что на Стек навязана структура Стека тоесть для данных в нем поддерживается принцип LIFO. В то время как на Кучу такие ограничения не накладываются.

Динамические массивы

Динамическим называется массив, размер которого может изменяться во время исполнения программы. Возможность изменения размера отличает динамический массив от статического, размер которого задаётся на момент компиляции программы.

Связные списки

Свя́зный спи́сок — базовая динамическая структура данных в информатике, состоящая из узлов, содержащих данные и ссылки («связки») на следующий и/или предыдущий узел списка.

Бинарные деревья

Двои́чное(бинарное) де́рево — иерархическая структура данных, в которой каждый узел имеет не более двух потомков (детей). Двоичное дерево является упорядоченным ориентированным деревом.

Использование бинарных деревьев в прикладных задачах

Бинарные деревья имеет смысл разделять на два "вида" в контексте их применения:
  • Представляющие собой структуры древовидные данных
  • Применяемые как способ доступа к другим структурам

Двоичное дерево поиска

Двоичное дерево поиска — двоичное дерево, для которого выполняются следующие дополнительные условия (свойства дерева поиска):
  • оба поддерева — левое и правое — являются двоичными деревьями поиска;
  • у всех узлов левого поддерева произвольного узла X значения ключей данных меньше либо равны, нежели значение ключа данных самого узла X;
  • у всех узлов правого поддерева произвольного узла X значения ключей данных больше, нежели значение ключа данных самого узла X.
